# User Management Service

## Application Type
REST API Microservice

## Features
* CRUD of users
  
* User Authentication
  
* ROLE BASED ACCESS CONTROL

* Swagger for API Documentation

## Technology Stack
* Java
* Spring Boot
* Maven 
* Hibernate JPA
